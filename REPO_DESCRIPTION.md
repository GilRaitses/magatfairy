## Full Description

```
mat2h5 - MATLAB to H5 Conversion Tool

A standalone, user-friendly tool for researchers who use MATLAB and MAGAT Analyzer to process optogenetic genotype experiment data. Converts experiment sets (ESETs) from MATLAB format to H5 files compatible with Python analysis pipelines.

Features:
- Interactive setup and conversion workflow
- Automatic MAGAT codebase cloning (or use existing)
- Batch processing of multiple ESET directories
- Complete validation framework included
- Self-contained (includes MAGAT Bridge code)
- No external dependencies beyond MATLAB and Python packages

Perfect for researchers who want to:
- Convert MAGAT experiment data to H5 format
- Process root folders containing ESET directories
- Export data for Python-based analysis workflows
```
